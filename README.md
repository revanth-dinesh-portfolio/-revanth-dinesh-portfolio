
# revanth-dinesh-portfolio

This is a minimal React app (Create React App style) containing a ready-to-deploy personal portfolio homepage for **Revanth Dinesh**.

How to use
1. Extract the ZIP.
2. (Optional) Edit `src/App.js` to customize content or images.
3. Run locally:
   - `npm install`
   - `npm start`
4. Build for production:
   - `npm run build`
5. Deploy:
   - Connect this repository to Vercel or Netlify and deploy.
   - Vercel will detect React and run `npm install` + `npm run build` automatically.

Files included:
- public/index.html
- src/index.js
- src/App.js (your homepage)
- src/App.css (basic styles)
- package.json

Note: This setup uses `react-scripts` for simplicity and broad compatibility with hosting platforms.
